# Code of Conduct

## Our Commitment

We are committed to maintaining a respectful, professional, and inclusive environment for everyone interacting with the Forge Kit project.

This Code of Conduct applies to:
- Issues
- Pull requests
- Discussions
- Any other project-related interactions

---

## Expected Behavior

Participants are expected to:
- Be respectful and considerate
- Assume good intent
- Provide constructive feedback
- Focus on technical merit
- Communicate professionally

---

## Unacceptable Behavior

Unacceptable behavior includes:
- Harassment or discriminatory language
- Personal attacks
- Trolling or inflammatory comments
- Disruptive or bad-faith participation

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

They may:
- Remove content
- Lock discussions
- Decline contributions
- Restrict participation

---

## Scope

This Code of Conduct applies within project spaces and when representing the project in public spaces.
